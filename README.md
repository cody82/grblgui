## Requirements
* Java 1.6
* RXTX (Ubuntu: apt-get install librxtx-java)
* OpenGL 2.0

## Usage
### Commandline arguments
Start the Program: java -jar grblgui.jar <your g-code file> <arduino port>
Example: java -jar grblgui.jar box.g /dev/ttyACM0
